### Usage

开发 React Native 前期的装备工作较多, 需要查阅官方文档!

https://facebook.github.io/react-native/docs/getting-started.html

这份文档打算梳理一下 React Native 的学习路径和常见问题, 方便前端开发者快速上手.

[查看目录](SUMMARY.md).

还没有涉及:

* 手势
* 数据存储
